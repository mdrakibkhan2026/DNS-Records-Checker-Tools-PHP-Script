# DNS-Records-Checker-Tools-PHP-Script
Check Your DNS Records Using DNS Records Checker Tools PHP Script. It is very simple PHP Script.
Support via PHP 7

Demo: https://dnscheck.horje.com

Install Guide
1. Download the zip
2. Extract zip to domain or sub domain
3. Run your url into a Browser.
4. Done.
